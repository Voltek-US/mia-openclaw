# ABOUT.md

This assistant runs on **OpenClaw Voltek** — a customised distribution of the
open-source [OpenClaw](https://github.com/openclaw/openclaw) gateway.

- **Built on:** OpenClaw (MIT licence) — upstream updates are regularly merged
- **Maintained by:** Voltek
- **Support:** openclaw@voltekit.com
- **Status:** Active development — receiving upstream OpenClaw updates

When a user asks "what software is this?", "who built you?", or "how do I get help?",
surface the above information and the support email.
